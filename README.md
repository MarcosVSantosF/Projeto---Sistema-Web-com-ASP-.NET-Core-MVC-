# Projeto - Sistema Web com ASP.NET Core MVC

Este projeto é um sistema web desenvolvido com ASP.NET Core MVC. O objetivo do projeto é criar uma aplicação robusta e escalável para gerenciamento de vendas. A aplicação permite cadastrar vendedores, departamentos e registrar vendas, além de visualizar relatórios detalhados. O sistema foi projetado para ser fácil de usar e oferece uma interface intuitiva, utilizando várias tecnologias modernas para garantir performance e segurança.

## Tecnologias Utilizadas no Projeto 🚀

- **ASP.NET Core MVC** 🌐
  - Framework para construção de aplicações web robustas e escaláveis.
  - Suporte para Model-View-Controller (MVC).

- **C#** 🖥️
  - Linguagem de programação principal para o desenvolvimento do projeto.
  - Suporte para programação orientada a objetos.

- **Entity Framework Core** 🗃️
  - ORM (Object-Relational Mapping) para trabalhar com banco de dados.
  - Facilita a manipulação de dados de forma eficiente.

- **MySQL** 🛢️
  - Sistema de gerenciamento de banco de dados relacional.
  - Armazenamento e gerenciamento dos dados da aplicação.

- **Bootstrap** 🎨
  - Framework CSS para design responsivo e estilização da aplicação.
  - Facilita a criação de layouts modernos e responsivos.

- **HTML5 & CSS3** 📝
  - Linguagens de marcação e estilo para estruturação e apresentação do conteúdo web.

- **JavaScript** ⚙️
  - Linguagem de script para adicionar interatividade à aplicação.
  - Utilização de bibliotecas como jQuery para facilitar o desenvolvimento.

- **Git** 🗂️
  - Sistema de controle de versão distribuído.
  - Facilita o versionamento e colaboração no código fonte do projeto.

- **Visual Studio** 🛠️
  - Ambiente de desenvolvimento integrado (IDE) usado para codificação, depuração e teste.

- **GitHub** 🌍
  - Plataforma de hospedagem de código-fonte e controle de versão.
  - Utilizada para colaboração e versionamento do projeto.
